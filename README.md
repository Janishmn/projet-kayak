# Weather & Hotels Scraper

Ce script Python scrape des données météo pour 35 villes françaises, puis récupère des informations sur les hôtels de ces villes sur Booking.com. Il génère plusieurs fichiers CSV avec :
- Les coordonnées des villes (`cities_geoloc.csv`)
- Les prévisions météo sur 5 jours et un score météo (`weather_data.csv`)
- Les 5 meilleures villes par score météo (`top_cities.csv`)
- 5 hôtels par ville (`top5_hotels.csv`)
